# Стоматологический бот для Telegram

Бот для автоматизации записи в стоматологическую клинику.

## Функционал
- Запись на прием к врачу
- Просмотр свободных слотов
- FAQ по клинике
- Напоминания о приеме
- Интеграция с Google Sheets
